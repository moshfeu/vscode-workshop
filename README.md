# vscode-workshop

## Installation

1. Install vscode (daaaa)
2. clone
3. run `yarn package` - a file named `vscode-workshop-0.0.1.vsix` will be created
4. Right click on the file and select `Install Extension VSIX`
5. In the prompt, fill your name

## How to use?

Hit <kbd>⌘</kbd> <kbd>↩</kbd>
